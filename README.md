# Bug Fix Challenge

## Goal

Fix one of the broken Python scripts in `scripts/` and submit a Pull Request.

## Steps

1. Fork this repository.
2. Pick a script folder that is not already assigned (list is below).
3. Fix the script following the coding conventions (PEP8).
4. Run `python3 tests/run_all_tests.py` to make sure your fix passes.
5. Commit and push your changes.
6. Open a Pull Request with title: "Fix script\_<number>".

## Rules

- Do NOT rename folders.
- Do NOT change script names.
- Do NOT modify others' work.
- One script per person unless explicitly allowed.
- Your code must run without errors.
